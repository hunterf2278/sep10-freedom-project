## Skills:
The two main skills I have learned when starting this blog and tinkering with my tool is being brave and trying new things, and to pay attention to my code. 

### Being Brave:
I have always hated tinkering in my IDE and I was very annoyed when my teacher saif we would need to do it for  part of the Freedom project. I never liked it because tinkering mean't I would have to do it on my own, there wouldn't be a teacher or friends to immediately ask for guidelines or help. Also, If I did it on my own I was more likely to make  mistakes that I might not be able to fix, then those mistakes might mess up my website. However, I realized while tinkering that its acutally  more effiecent to tinker on my own because I learned different what commads and code do, and how to do things for myself.

### Focusing:

Whenever im coding there have been a couple of instances where im so focused on what I want to code, how much code I should have or need, that I forget all about the little things. I was so focused on figuring out how to add more 3d shapes and add more code in order to make my scene more interesting, I forgot that sometimes whenever your changing one number in the dimensions of the shapes, you might need to change the  other numbers so that it's proportionate. This reminded me that I need to pay close attention to all the code I have first and make sure I understand it, before I try an add more.

Now that I have chosen my tool, Aframe, and started tinkering and learning more about it, there are so many oppurtunities that await me. I have the ability to deisgn my product through 3d shapes and this will make my website even more exciting!
